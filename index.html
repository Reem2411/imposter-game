üéÆ How to Play

  1. Create Session: One player creates a session and shares the code
  2. Join Players: Others join using the session code
  3. Start Game: Host starts when 3+ players are ready
  4. Find Imposter: All players except one know the secret word
  5. Vote: Players vote to eliminate who they think is the imposter
  6. Results: See if the imposter was caught!
